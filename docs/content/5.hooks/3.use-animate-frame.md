`useAnimateFrame` 每个动画帧运行一次回调。

```vue
<script setup>
import { useAnimationFrame } from 'motion-v'

useAnimationFrame((time) => {
  dom.value.style.transform = `rotateY(${time}deg)`
})
</script>
```

## 使用
<ComponentPreview name="animate-frame" />
回调函数提供了两个参数：

- `时间`：自回调首次调用以来总的毫秒数。
- `delta`：自上次动画帧以来的时间（毫秒）。

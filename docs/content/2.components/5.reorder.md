## 重新排序组属性



::PropsTable
---
data:
- name: axis
  default: "y"
  type: "'x' | 'y'"
  description: 重新排序的轴。要使两个轴都可拖动，设置 `<ReorderItem drag />`
- name: onUpdate:values
  default: "-"
  type: "(newOrder: V[]) => void"
  description:  拖动结束时的回调函数，用于触发新值顺序。
- name: values
  default: "[]"
  type: "V[]"
  description: 最新值状态。
---
::
